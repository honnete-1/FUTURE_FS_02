#  Mini E‑Commerce Platform

This project is a **Mini E‑Commerce Platform** developed as **Task 2** for the  
[Future Intern Full Stack Web Development Internship](https://futureinterns.com/full-stack-web-development-task-2/).

##  Live Demo

 [View Website](https://futurefs02-iota.vercel.app/)

---

##  Task Overview

The objective of this task was to build a **functional mini e‑commerce platform**
that demonstrates core full‑stack concepts including frontend UI design, backend
logic, and database interaction.

The application allows users to browse products and interact with the platform
through a clean and responsive interface.

---

## Technologies Used

### Frontend
- React
- React Router DOM
- CSS for styling
- Axios / Fetch API for HTTP requests

### Backend
- Node.js
- Express.js
- MySQL database
- CORS & dotenv

### Deployment
- Frontend: Vercel  
- Backend: Render  
- Database: MySQL

---

##  Features

- Product listing and display
- Dynamic rendering of product data
- Client–server communication via REST APIs
- Responsive user interface
- Structured and scalable project architecture
